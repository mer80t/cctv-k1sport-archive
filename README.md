# cctv-k1sport-archive

本项目用于归档和发布多个独立 HTML 页面。这些页面内容来源于不同渠道，不针对任何单个域名或具体网站。仓库旨在保存历史快照，方便后续查阅与回放。

## 项目简介

cctv-k1sport-archive 是一个静态 HTML 页面归档仓库。仓库内容为独立、自包含的 HTML 文件，每个文件代表一个独立的归档页面。页面内容可能包含文本、图片、链接等元素，但所有资源均内嵌或通过相对路径引用，不依赖外部特定服务。

## 目录说明

```
/
├── README.md              # 本文件
├── index.html             # 主索引页面，展示归档列表
├── pages/                 # 存放所有归档 HTML 页面
│   ├── 2024-01-01.html    # 示例归档文件
│   ├── 2024-01-02.html
│   └── ...
└── assets/                # 公共资源（如果有）
    ├── styles.css
    └── scripts.js
```

- **index.html**：仓库首页，提供归档页面的导航和简要说明。
- **pages/**：所有归档 HTML 页面存放于此，文件命名通常包含日期或标识符。
- **assets/**：可选的公共样式、脚本或图片资源（如果页面间共享）。

## 页面归档说明

- 每个归档页面均为独立 HTML 文件，可直接在浏览器中打开查看。
- 归档内容可能包含文字、表格、图片（Base64 内嵌或相对路径引用）等。
- 页面不依赖任何外部网络资源，确保离线可查看。
- 归档文件命名建议采用 `YYYY-MM-DD-描述.html` 格式，以便按时间排序。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面。
- 欢迎提交 Issue 或 Pull Request 补充或修正归档内容。
- 新增页面请遵循以下规则：
  - 文件放置于 `pages/` 目录下。
  - 确保页面为独立 HTML，不引用外部域名资源。
  - 如有必要，更新 `index.html` 中的导航列表。
- 仓库不保证归档内容的完整性、准确性和时效性，仅作存档用途。
